## 🚀 How to Start

### Option 1: Double-click the batch file
Simply double-click `start-dev.bat` to start the development server.

### Option 2: Use command line
```bash
npm run dev
```

Then open: http://localhost:3000

## 📝 Next Steps

### 1. Add Real Images
Replace image placeholders with actual photos:
- Put images in the `public/images/` folder
- Update components to use real images

### 2. Customize Content
- Edit room prices in `pages/rooms.js`
- Update package details in `pages/packages.js`
- Add more testimonials in `pages/index.js`

### 3. Deploy to Production

**Vercel (Easiest):**
```bash
npm install -g vercel
vercel
```

**Netlify:**
- Connect GitHub repository
- Build command: `npm run build`
- Publish directory: `.next`

**Static Export:**
```bash
npm run build
npm run export
```
Upload the `out` folder to any hosting.

## 🔧 Troubleshooting

**Port already in use?**
The default port is 3000. If it's busy, Next.js will automatically use 3001.

**Build errors?**
Run: `npm install` to reinstall dependencies.

**Need help?**
Check the README.md file for detailed documentation.

# Rana Guest House Website

A modern website for **Rana Guest House** located in **Baragaon, Joshimath, Uttarakhand**.

Built using **Next.js** to showcase rooms, nearby attractions, and contact information for guests.

## Features

- Responsive design
- Rooms & pricing information
- Nearby tourist places
- Google Maps location
- WhatsApp & call contact options

## Tech Stack

- Next.js
- React
- CSS
- Lucide Icons

## Run Locally

```bash
npm install
npm run dev
